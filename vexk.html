Vexk App

This is the Bitcoin Exchange App on the iOS platform.
Trading coins conveniently within the application.


When you access the app for the first time,
you use the API given by the server to expose the announcement pop-up and show the basic coin list.


Login is treated as a web view, You can link it with the ID of each platform.


The server adjusts the index to meet the customer's request and exposes each coin data to the user.


It's a practically tradable tab,The conditions of each minimum transaction price are at stake. You can also check the amount in real time.


This is where you can see the user's information, and it also comes with translations of three languages. (Batnam, China, and Korea)
In the case of membership withdrawal, it is also carried out here.
