Vexk App

これはiOSプラットフォーム上のBitcoin Exchange Appです。
アプリケーション内で便利にコインを取引できます。


初めてアプリにアクセスするときは、
サーバから提供された API を使用して、アナウンス ポップアップを表示し、基本的なコイン リストを表示します。


ログインは Web ビューとして扱われます。各プラットフォームの ID とリンクできます。


サーバーは顧客の要求に合わせてインデックスを調整し、各コインデータをユーザーに公開します。


これは実質的に取引可能なタブです、それぞれの最低取引価格の条件が関係しています。リアルタイムで金額を確認することもできます。


ここでユーザーの情報を見ることができ、3つの言語の翻訳も提供されます。（ベトナム、中国、韓国）
